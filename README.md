# ☀️ Solar Panel Fault Detection Using Convolutional Neural Networks (CNN)

## 📖 Overview

This project presents a deep learning-based approach for automatic fault detection in photovoltaic (PV) systems using a Convolutional Neural Network (CNN).

The model learns from electrical measurements collected from solar panels and classifies different fault conditions automatically. This helps improve the reliability, maintenance, and performance of photovoltaic power systems.

---

## 🚀 Features

- Data preprocessing and normalization
- Sliding Window feature extraction
- CNN-based multi-class classification
- Label Encoding & One-Hot Encoding
- Train/Test data splitting
- Model training using TensorFlow/Keras
- PCA visualization (3D)
- Confusion Matrix
- Accuracy and performance evaluation
- Model saving and loading
